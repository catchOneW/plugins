# @umijs/plugin-lodash

> @umijs/plugin-lodash.

See our website [@umijs/plugin-lodash](https://umijs.org/plugins/plugin-lodash) for more information.

## Install

Using npm:

```bash
$ npm install --save-dev @umijs/plugin-lodash
```

or using yarn:

```bash
$ yarn add @umijs/plugin-lodash --dev
```
